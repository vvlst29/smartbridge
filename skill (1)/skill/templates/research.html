Research Workflow

Crop-environment analysis for agricultural planning.

1. Data Collection and Analysis

The project uses crop recommendation records containing N, P, K, temperature, humidity, pH, rainfall, and crop labels. Exploratory analysis supports univariate, bivariate, and multivariate understanding.

2. Data Preprocessing

Missing values are checked, outliers can be inspected with IQR and boxplots, and the dataset is split into training and testing sets for reliable evaluation.

3. Model Building

The training script compares Logistic Regression and Random Forest models. The best-performing classifier is saved as models/model.pkl.

4. Reports and Policy Use

Predictions can support crop planning, sustainable water and fertilizer use, agricultural research, and evidence-based policy decisions.
